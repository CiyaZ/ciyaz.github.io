# 地理定位

地理位置API能够通过IP定位用户的经纬度。

```javascript
if ('geolocation' in navigator) {
  console.log('地理服务可用');
  navigator.geolocation.getCurrentPosition(function (pos) {
    // 获取纬度
    console.log(pos.coords.latitude);
    // 获取精度
    console.log(pos.coords.longitude);
  });
} else {
  console.log('地理服务不可用');
}
```
